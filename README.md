# homebrew-lore-book

Homebrew tap for [lore-book](https://github.com/CptPlastic/lore-book) — the spellbook for your codebase.

## Install

```bash
brew tap CptPlastic/lore-book
brew install lore-book
```

## Usage

```bash
lore --help
```

## Upgrade

```bash
brew upgrade lore-book
```
